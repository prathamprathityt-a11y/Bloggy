Bloggy

Bloggy is a modern blogging and storytelling website where users can share articles, blogs, personal stories, and creative content in a clean and responsive interface.

Features

- Responsive modern UI
- Blog and article layout
- Interactive JavaScript functionality
- Clean design with custom CSS
- Fast and lightweight website
- Beginner-friendly project structure

- JavaScript
